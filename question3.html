Q3.Explain the difference between undefined and not defined variables in JS using HTML page.

  • Undefined
  • not defined

Undefined

It is a JavaScript keyword that has a special meaning. Everything which gets a space in memory
will contain undefined until we assign a value to that memory space.
var x;
console.log(x); ---> Here varaible x is undefined type.

not defined

In JavaScript, it is one of the reference errors that JavaScript will throw when someone
accesses the variable which is not inside the memory heap.
console.log(y); ---> Here varaible y is not defined type.
